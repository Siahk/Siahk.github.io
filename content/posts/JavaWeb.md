# 1 JavaWeb

## 1.1 基本概念

Web 开发：
- Web：网页
- 静态 Web
    - html、css
    - 提供给所有人看的数据都不会发生变化。
- 动态 Web
    - 技术栈：Servlet/JSP、ASP、PHP

**定义：** Java Web 应用由一组 Servlet、HTML 页、类、以及其它可以被绑定的资源构成，它可以在各种供应商提供的实现 Servlet 规范的 Servlet 容器中运行。
